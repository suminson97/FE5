## 기능

### 진열대 상품 생성

- 상품 데이터 파일 세팅
- 데이터 불러오기
- 상품 요소 생성
- 품절 상품 표시

### 2. 장바구니

- 진열대 상품 클릭 시
  - **예외** count(재고) 없을 시, 안내창
  - **예외** 잔액 부족 시, 안내창
  - 장바구니에 있을 시, 수량 증가
  - 없을 시 요소 생성
  - 재고 변경
  - 재고 소진 시, 품절 표시
- 장바구니 상품 클릭 시
  - 1개 : 요소 삭제
  - 2개 이상 : 수량 감소
  - 재고 변경
  - 재고 소진 시, 품절 표시

### 2. 장바구니

- 진열대 상품 클릭 시
  - 장바구니에 있을 시, 수량 증가
  - 없을 시 요소 생성
